# Skill 候選清單（2026-05-20）

**已建立**：
- `secondbrain-research-digest` — 將 Secondbrain 內的資料整理、查詢、比較與決策需求，輸出成 Obsidian-ready Markdown 筆記與結構化資料。

**建議接下來建（按優先度）**：
1. `secondbrain-weekly-review` — 每週整理 `Clippings/`、`每日筆記/`、`知識庫/` 的新增內容，補出可行動的知識整理清單；適合每週固定重整。
2. `decision-comparison-table` — 專門把工具、課程、方案、工作流比較成決策表與建議排序；適合查詢/決策頻率高的任務。
3. `obsidian-note-refiner` — 將零散草稿整理成正式 Obsidian 筆記，保留原語氣並補 frontmatter、段落與待辦；適合筆記整理。
4. `teaching-material-digest` — 從教學素材、影片筆記或剪藏中抽出課程可用段落、提問、案例與活動設計；適合教學準備。
5. `project-cockpit-updater` — 專門更新 `專案庫/<專案名稱>/專案工作流程.md` 的狀態、下一步與阻礙；適合多專案管理。

**規則**：
- 每週先用 1 個 skill，不要一次養太多。
- 每次實際使用後，記錄哪裡不順，再回來微調 `description`、流程或模板。
- 如果某個 skill 連續一個月都沒被使用，應該合併、改名或刪除。
