# Arry 助手行動清單

## 作業 1：餵 Arry 助手你的過去作品

目標：讓 Arry 助手未來寫作、整理、教學輸出時，先讀你的範例，而不是憑空想像風格。

請把 5-10 份代表性作品放入：

- Facebook 貼文：`200_Reference/writing-samples/social/`
- Email、合作邀約、客戶溝通：`200_Reference/writing-samples/emails/`
- 書籍讀後心得：`200_Reference/writing-samples/articles/`
- Podcast 讀後心得或口語稿：`200_Reference/writing-samples/scripts/`

## 作業 2：整理常用模板

請把常用格式放入：

- 一般模板：`200_Reference/templates/`
- Email 模板：`200_Reference/templates/email-templates/`

## 作業 3：挑 3 個最重複的流程

請列出最近最常重複做的 3 件事，之後可評估是否升級成 Codex 全域 skill。

範例：

- 每週知識重整
- 資料研究摘要變成 Facebook 貼文
- 書籍或 Podcast 筆記整理成讀後心得
- 客戶溝通 Email 草稿與回覆模板

## 作業 4：重開 Codex

重新開啟 Codex 對話或重啟 Codex App，讓新的 `arry-assistant` skill 與全域規則生效。
