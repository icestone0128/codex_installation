# 05-第二大腦設定指南

## 目標

把第二大腦整理成可長期使用的工作系統：剪藏、模板、知識庫、每日筆記、每週重整與專案駕駛艙。

## 已建立項目

- `Clippings/`
- `知識庫/`
- `創作庫/`
- `每日筆記/`
- `Templates/`
- `專案庫/`
- `知識庫/index.md`
- `知識庫/log.md`
- 每日筆記與知識庫模板

## Web Clipper 設定

Obsidian Web Clipper 要確認：

- vault 指向 `secondbrain`
- clipping 預設資料夾可用 `Clippings/`
- Desktop Obsidian 已開啟正確 vault

## 自動化

已建立 Obsidian 每週知識重整自動化：

- 名稱：`Obsidian 每週知識重整`
- 時間：每週日早上 6:00
- 工作區：`secondbrain`

## 踩坑修正

- `Vault not found` 多半是 Web Clipper 的 vault 參數錯誤，不是檔案內容錯。
- Web Clipper 成功剪藏後，若 Desktop 看不到，先確認 Desktop vault 與 Web Clipper vault 是同一個。
- 每週重整寫到 `每日筆記/` 或專案駕駛艙，不要污染全域 `AGENTS.md`。
- `AGENTS.md` 寫固定規則；`log.md` 與駕駛艙寫事件與進度。
